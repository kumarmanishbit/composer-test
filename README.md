## Magento's extension for connector

#### Guide to do the development:

Step 1 ) Clone this repository.

git clone git@orahub.oci.oraclecorp.com:omc-apps-dev/magento-extension.git bronto

Step 2 ) After development generate the distribution package by using below command: 
```
  php ci/build.php --environment prod --version 1.0.0
```

This command will generate deliverable under build folder.
